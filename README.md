# agent-skills

Agent skills from [agentspace.so](https://agentspace.so/?utm_source=github&utm_medium=readme&utm_campaign=agent-skills) — ready-to-install skills for Claude Code, Cursor, and other AI coding agents.

## Skills

| Skill | Description |
|---|---|
| [`gpt-image-2`](./gpt-image-2) | Generate images with GPT Image 2 via your existing ChatGPT Plus/Pro subscription — no OpenAI API key, no per-image billing. Runs through the local `codex` CLI. |

## Install

```bash
npx skills add agentspace-so/agent-skills --skill <skill-name> -g
```

Example:

```bash
npx skills add agentspace-so/agent-skills --skill gpt-image-2 -g
```

Or browse on [skills.sh/agentspace-so/agent-skills](https://skills.sh/agentspace-so/agent-skills).
